# Syntax

## Input/Output

### Input

```
INPUT variable
```

### Output

```
OUTPUT variable
```

## Selection Constructs

```
IF condition THEN
	action
ELSE IF condition THEN
	action
ELSE
	action
ENDIF
```

## Iteration Constructs

### For loops

```
FOR variable to condition //count 1 to 5, can be used without prior declaration
	action
NEXT variable //to increment count
```

### While loops

```
WHILE condition //is true
	action
ENDWHILE
```

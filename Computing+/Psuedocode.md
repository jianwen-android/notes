# Syntax

## I/O

### Input

```
INPUT variable
```

### Output

```
OUTPUT variable
```

## If statements

```
IF condition THEN
	action
ELSE IF condition THEN
	action
ENDIF
```

## Loops

### For loops

```
FOR variable to condition //count 1 to 5, can be used without prior declaration
	action
NEXT variable // to increase count usually
```

### While loops
```
While condition //is true
	action

```
